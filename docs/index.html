Bienvenido al Tutorial de Robótica

Este tutorial está diseñado para ayudarte a aprender las herramientas necesarias para el desarrollo de proyectos en robótica.

Usaremos herramientas como Git, ROS, SolidWorks y LaTeX para una integración completa del flujo de trabajo.
